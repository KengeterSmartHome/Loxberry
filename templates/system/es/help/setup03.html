Configuración de Red

Tipo de Red: LAN / Red Cableada o WLAN / Red Inalámbrica.
Elija el tipo de conexión física a su Loxberry.

Dispositivos de Red: Seleccione el dispositivo que se utilizará para establecer un enlace de red.

WLAN (SSID): Introduzca el nombre de su WiFi / WLAN.

Contraseña WLAN: Introduzca su contraseña WiFi / WLAN.

Tipo de Direccionamiento: Seleccione el método para obtener una dirección IP. Tanto la gestión automática (DHCP) como la gestión manual (IP estática)

Dirección IP: En caso de gestión IP estática, introduzca aquí una dirección IP. Una dirección IP tiene el formato xxx.xxx.xxx.xxx, por ejemplo 192.168.1.100

Máscara de Subred: En caso de gestión IP estática, introduzca la Máscara de subred IP de su red local. En muchos casos es 255.255.255.0

Puerta de Enlace: En caso de gestión IP estática, introduzca la dirección IP de la puerta de enlace. En la mayoría de los casos, la IP de su Router/Fritz!Box.

DNS: En caso de gestión IP estática, introduzca la dirección IP del Servidor de nombres (DNS) En la mayoría de los casos, la IP de su Router/Fritz!Box.
