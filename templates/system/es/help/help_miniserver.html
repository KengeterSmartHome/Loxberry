Usar DNS Cloud: Marque la casilla para utilizar el servicio Loxone Cloud DNS para llegar al Miniserver desde el exterior.

Dirección IP o Host: La dirección IP o el nombre de host de Loxone Miniserver. Por favor, asegúrese de que el Miniserver es accesible todo el tiempo utilizando este IP/Host.

Puerto: El Número de Puerto para llegar a la Interfaz Web de los Miniservers. Generalmente Puerto 80.

Dirección MAC: La dirección MAC del Miniserver si utiliza el servicio de DNS de la nube de Loxone. Puede encontrar la dirección MAC en el administrador del Miniserver o en el Loxone Config. Es una cadena de 12 caracteres que comienza con EEE000xxxxxx o 504Fxxxxxxxx.

Puerto FTP Publico: El puerto FTP público del Miniserver si utiliza el servicio de DNS de la nube de Loxone. Puede administrar este puerto en su enrutador (por ejemplo, Fritz! Box). El puerto interno debe mantenerse en 21. De lo contrario, el Fritz! Box Firewall/Application Layer Gateway no puede identificar el tráfico como TRafico FTP. Lo mismo puede suceder para otros modelos de Router.

Usuario del Miniserver: Nombre de usuario de una cuenta de administrador del Miniserver. En la mayoría de casos es "admin".

Contraseña del Miniserver:: La contraseña asociada a la cuenta de administrador anterior.

Nota / Enlace: Un campo para usar libremente, colocar la localización, nombre, etc..
Si introduce un vínculo (URL), un botón se mostrará automáticamente para abrir el vínculo en una ventana nueva.

Nombre: Para asignar un nombre al Miniserver - es utilizado por algunos plugins para identificar el Miniserver.
